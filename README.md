# awaitful

Earn from your AI coding agent's wait time. When your agent (Claude Code and others) is thinking,
Awaitful shows a single tasteful, clickable sponsored line — and you earn a share of the revenue. It
only detects *that* an agent is thinking and *for how long*; it never reads your code, prompts,
files, or the agent's output.

Get started at **https://awaitful.com**.
